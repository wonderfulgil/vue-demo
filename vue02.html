循环绑定数据v-for

1.数组

基本用法

v-for 指令需要以 item in items 形式的特殊语法， items 是源数据数组并且 item 是数组元素迭代的别名。

  • {{list}}

当前项的索引

在 v-for 块中，我们拥有对父作用域属性的完全访问权限。 v-for 还支持一个可选的第二个参数为当前项的索引,如：v-for='(item, index) in items'。

  • {{ index }}--{{list}}

of分隔符

可以用 of 替代 in 作为分隔符,如：v-for='(item, index) of items'：

  • {{ index }}--{{list}}

2.Json数组

基本用法

v-for 指令需要以 item in items 形式的特殊语法，获取内容项：item.name 。

  • 姓名：{{list.name}} 性别：{{list.sex}} 年龄：{{list.age}}

当前项的索引

在 v-for 块中，我们拥有对父作用域属性的完全访问权限。 v-for 还支持一个可选的第二个参数为当前项的索引,如：v-for='(item, index) in items'。

  • {{ index }}--姓名：{{list.name}} 性别：{{list.sex}} 年龄：{{list.age}}

of分隔符

可以用 of 替代 in 作为分隔符,如：v-for='(item, index) of items'：

  • {{ index }}----姓名：{{list.name}} 性别：{{list.sex}} 年龄：{{list.age}}

3.json对象迭代 v-for

可以用 v-for 通过一个对象的属性来迭代。如：v-for='(list, index) of object'：

基本用法

  • {{ index }}--{{list}}

提供第二个的参数为键名

如：v-for="(value, key) in object"
  • {{ key }} -- {{ value }}

第三个参数为索引：

如：v-for="(value, key, index) in object"
  • {{ index }}--{{ key }} : {{ value }}
